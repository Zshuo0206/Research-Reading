## 产品目标

验证个人研究者使用单用户 Web 工作台进行方法学习时，能否通过学习目标、问题驱动、主动作答和证据回溯获得比普通 PDF 摘要更有帮助的阅读体验。

## V1.0 产品模式

- 快速问答模式：保留问题计划确认/修改/拒绝，以及至少一个带证据回答的确认闭环。
- 引导式学习模式：V1.0 核心路径；目标输入 → 候选方向 → 固定三阶段路线 → 第一阶段逐题学习 → 阶段总结。

## 纳入范围

- 单用户本地 Web 工作台和一篇文本型、可提取内容的科研论文 PDF。
- 学习目标自由文本输入，生成 2–3 个候选学习方向并由用户选择。
- 固定三阶段路线；仅开放“理解内容”，其余阶段显示“即将开放”。
- 一次生成 3–7 道开放式理解题，按顺序逐题展示。
- 用户自由文本作答或“不会/跳过”；提交后获得简短点评、论文内参考答案和页码/稳定位置证据。
- 参考答案可见时修改当前最新版回答，主动进入下一题，完成后生成简短阶段总结。
- 快速问答模式继续保留；Mock 与 BYOK 遵守同一结构化输出和证据校验边界。

## 排除范围

- OCR、扫描 PDF、复杂表格/公式/图片理解和多篇论文联合学习。
- 分析评价、迁移应用、动态难度、追问、外部知识检索、选择题、评分和掌握等级。
- 完整学习历史、回答版本历史、复杂学习报告、推荐阅读和额外练习。
- 资产库、Markdown/Excel 导出、协作、权限体系、插件、移动端、远程部署和 UI 精修。

## 产品验收标准

1. 从空库导入许可的文本型 PDF 后，用户无需编辑 URL 或调用 API 即可进入学习流程。
2. 用户可以输入学习目标、选择候选方向并看到三阶段路线；未开放阶段不可进入。
3. 第一阶段完成 3–7 道逐题开放题；每题支持阅读、作答/跳过、点评、参考答案、证据、修正和主动下一题。
4. 证据可追溯到论文版本、页码/稳定文本位置和原文片段，并区分论文事实、作者主张和 Agent 推断。
5. 阶段完成后展示简短总结；不产生 V1.0 明确排除的评分或复杂报告。
6. 快速问答另行完成问题和回答确认闭环，未确认内容不进入正式研究资产。
7. Mock 路径可在 CI 和无网络环境确定性运行；BYOK 仅用于本地人工验收，密钥不持久化。

## 技术与治理边界

- 第一轮已集成的 SQLite/Job、文本 PDF 提取、canonical page text、Evidence 坐标校验和 Mock Gateway 可复用。
- 引导式学习涉及公共契约、状态机和可能的数据库字段；必须先审查 [RFC-W1-002](../rfcs/RFC-W1-002-guided-learning-v1-product-alignment.md)。
- 任务 `T-W1-005`、`T-W1-006`、`T-W1-011` 更新为 V1.0 入口，但仍保持 `DRAFT/LOCKED`；文档更新不等同于 Gate C 编码授权。
- 真实外部模型调用不进入普通 CI；平台只支持本地 `127.0.0.1` 边界。
