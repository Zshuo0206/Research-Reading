# Definition of Done

任务只有同时满足以下条件才可标记 `DONE`：

1. 所有验收条件满足。
2. 变更文件在任务 ownership 内。
3. 必要 RFC/ADR 已批准并关联。
4. Agent 已提交完整交接报告和可审计证据。
5. 责任 Agent 已完成局部检查，QA 已完成适用的独立质量检查，主控 Agent 已重新运行关键测试并记录结果。
6. 错误、边界、安全和可观测性要求已检查。
7. 文档、任务状态、devlog 和 CURRENT_STATE 已更新。
8. 集成分支可构建、可测试、可恢复。
9. 主控 Agent已确认没有被绕过的强制门槛，或已记录批准的例外。
10. 若属于 Wave 结束，主控已生成简化人类验收摘要；人类负责人不需要逐项复核工程细节。
