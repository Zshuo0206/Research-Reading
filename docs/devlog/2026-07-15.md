## Interface changes

新增或正式暴露的公共入口包括：

- `packages/contracts/wave1/guided-learning.v1.schema.json`；
- `packages/contracts/wave1/generated/guided-learning.v1.d.ts`；
- `packages/contracts/wave1/src/guided-learning.ts`；
- `SupportedContractVersion`；
- `validateGuidedLearningSessionConsistency`；
- `applyGuidedLearningEvent`；
- `GUIDED_LEARNING_STATES`；
- `GUIDED_LEARNING_TRANSITIONS`；
- `GUIDED_LEARNING_CONSISTENCY_ERROR_CODES`。

契约摘要：消息分为 `SESSION` 与 `COMMAND`；客户端使用严格写入类型；Evidence readiness
由服务端提供；failure 记录 `failed_operation`/`resume_state`；幂等记录包含
`request_fingerprint`；路线使用 canonical route，并由服务端判定完成条件。
本日志不复制整个 Schema。
